jQuery Plugins

Information

Very simple plugin that displays tabular data in json format in vertical or horizontal mode, translating th content specifying it in the props parameter.

Usage

$("#ajax-to-table-vertical").ajaxtotable({data: jvertical, orientation: 'v', props: props});
$("#ajax-to-table-horizontal").ajaxtotable({data: jhorizontal, orientation: 'h', props: props});
where json vertical and horizontal are like: 
var jvertical = {
	"id": "0001",
	"cmp": "HTC",
	"model": "HTC Desire",
	"price": "300 EUR"
};

var jhorizontal = [
	{
	"id": "0001",
	"cmp": "HTC",
	"model": "HTC Desire",
	"price": "300 EUR"
	},
	{
		"id": "0002",
		"cmp": "Apple",
		"model": "iPhone 4s",
		"price": "520 EUR"
	},
	{
		"id": "0003",
		"cmp": "Apple",
		"model": "iPad 3G",
		"price": "600 EUR"
	}
];
and props is like: 
var props = {
	"id": "ID",
	"cmp": "Company",
	"model": "Model",
	"price": "Price"
};

Parameters

Name Description
data Data to be displayed in json format
orientation Table orientation. Can be v or h meaning "vertical" or "horizontal"
props It is a json containing data to be displayed and related text to be shown in the th

Result

Vertical orientation

Horizontal orientation
