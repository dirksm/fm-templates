jQuery Plugins

Information

Retrieves photos from a public picasa web album for a specific user and album in json format. It creates an ul element and appends li thumbnails to it.

Usage

$("#photos").picasize({userid: '104458938478962853927', albumid: "5651671619363913953", imgsize: 640, thumbsize: '75c', startindex: 1, results: 15});

Parameters

Name Description
userid User's ID
albumid Album's ID
imgsize Size of enlarged images, in px
thumbsize Size of thumbnails. "c" means that they will be squared
startindex 1-based index of the first result to be retrieved
results Number of images to be retrieved
