jQuery Plugins

Information

Retrieves calendar events from a public google calendar in json format.

Usage

$("#calendar").calendarize({apikey: 'yourApiKey', calendarid: 'publicCalendarID', results: 3, mintime: null});

Parameters

Name Description
apikey Google Calendar API v3 key
calendarid Google Calendar ID of a public calendar
results Maximum number of events to retrieve
mintime Minimum date and time of the events in yyyy-MM-ddTHH:mm:ssZ format. Default is now()
