jQuery Plugins

Information

This is a collection of jQuery plugins I developed for personal purposes. I am not a javascript expert so they don't pretend to be an example of javascript programming, but they have been useful for me, so maybe they will be useful for someone else too.

For example, since I did not have any templating library such as Sitemesh, the menu on the left is built in a jQuery plugin, that centralizes the menu definition and is executed in every page of the website.

Usage

$("#sidebar-menu").menuize();

Every plugin is in the folder named "js", like for example the menuize.js one.
